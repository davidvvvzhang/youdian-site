# youdian-site

用于 App Store Connect 的营销与技术支持页面。

- 首页（营销）：`/`
- 技术支持：`/support/`

## 发布到 GitHub Pages

1. 创建公开仓库 `youdian-site`
2. 推送本目录代码
3. 在仓库 `Settings -> Pages` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main` / `/ (root)`

发布后：
- 营销网址：`https://davidvvvzhang.github.io/youdian-site/`
- 技术支持网址：`https://davidvvvzhang.github.io/youdian-site/support/`
